# 전기차 보조금 도우미

## 프로젝트 소개

**전국 161개 지역**의 전기차 보조금 지급 현황을 실시간으로 조회하는 Flutter 기반 크로스플랫폼 모바일 애플리케이션입니다. 

한국환경공단의 전기차 보조금 데이터를 자동으로 수집하여, 사용자가 지역별/차종별로 보조금 정보를 쉽게 비교할 수 있도록 제공하는 서비스입니다. 정부 기관과는 관련이 없는 비공식 정보 제공 앱으로, [무공해차 누리집](https://ev.or.kr) 등 공식 자료를 기반으로 합니다.

## 만든 배경

전기차 구매를 고려하는 소비자들이 가장 어려워하는 부분 중 하나가 복잡하고 다양한 보조금 제도를 이해하는 것입니다. 

- 국가 보조금과 지방자치단체 보조금이 각각 다른 기준으로 적용
- 차량 모델별, 지역별로 상이한 보조금 금액  
- 매년 변경되는 정책과 예산 소진 상황
- 여러 공식 사이트를 돌아다니며 정보를 찾아야 하는 번거로움

이러한 불편함을 해결하고, 전기차 구매 결정에 도움을 주고자 개발하게 되었습니다.

## 주요 기능

### 🔍 **지역별 보조금 조회**
- **전국 161개 지역**의 실시간 보조금 현황 제공
- 지역별 공고, 신청, 출고 현황 상세 표시
- 신청 잔여/출고 잔여 수량 및 진행률 시각화
- 상태별 필터링 (여유/신청가능/거의마감/마감)

### 📊 **차종별 보조금 비교**
- 실시간 자동완성 차량 모델 검색
- **157개 지역** 보조금 자동 비교
- 최고/최저 혜택 지역 TOP 5 제공
- 국고/지방 보조금 구성 내역 분석

### 📈 **데이터 시각화**
- **일별 출고 추이 그래프** (FL Chart 라이브러리)
- 최근 30일간 출고 잔여 변화 추이
- 인터랙티브 차트 및 툴팁 제공
- 통계 대시보드 (총 지역 수, 공고 수량, 신청률)

### 🎰 **스마트 기능**
- **행운의 지역 뽑기** (컨페티 애니메이션)
- 최대 절약 금액 계산
- AdMob 광고 통합 (배너/전면 광고)

## 아키텍처

### **Flutter 모바일 앱**
- **Flutter 3.32.4** + Dart ^3.8.1 크로스플랫폼 프레임워크
- **Provider 패턴**을 활용한 상태관리 및 아키텍처
- **FL Chart** 라이브러리를 통한 데이터 시각화
- **SharedPreferences**로 로컬 데이터 캐싱
- **HTTP 클라이언트**를 통한 REST API 통신
- **AdMob 광고** 통합 (google_mobile_ads ^5.1.0)

### **백엔드 시스템**
- **Node.js + Express.js** RESTful API 서버
- **MySQL 8.0** 데이터베이스 (utf8mb4 인코딩)
- **Swagger** API 문서화 자동 생성
- **CORS 정책** 및 보안 설정

### **데이터 수집 파이프라인**
- **Playwright** 브라우저 자동화 크롤링
- **Node-cron** 스케줄러 (매일 새벽 2시 실행)
- **Excel 파일 파싱** 및 데이터 검증
- **MySQL 데이터베이스** 자동 저장 및 인덱싱

### **앱 아키텍처 (Provider 패턴)**
```
UI Screens ↔ Provider (State) ↔ Services (API) ↔ Backend API
    ↓           ↓                    ↓              ↓
Widgets    Models            HTTP Client      MySQL DB
```
